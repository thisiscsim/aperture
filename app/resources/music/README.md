# Bundled music

Drop royalty-free audio tracks (`.mp3`, `.m4a`, `.wav`, `.ogg`) here to make them
selectable from the editor's Audio → Music → Library control.

These ship with the app, so only include tracks you have the right to distribute
(e.g. CC0 / public-domain). Tracks a user uploads themselves are copied into
their own `projects/<slug>/assets/` instead and are never added here.
